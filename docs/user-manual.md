# 懶人研究包 — 使用說明書

> 本工具為台股盤後輔助研究工具，資料僅供研究參考，不構成任何投資建議。

---

## 一、產品定位

懶人研究包回答三個問題：

| 層次 | 問題 |
|---|---|
| 首頁 | 今天市場焦點在哪？ |
| 族群頁 | 這個族群的強勢是否健康？ |
| 個股頁 | 這檔股票在族群裡扮演什麼角色？ |

核心流程：

```
首頁確認市場狀態 → 點入值得研究的族群 → 點入族群內的個股
```

---

## 二、快速上手

頁首提供兩種輔助入口：

| 入口 | 用途 |
|---|---|
| 新手導覽 | 逐步高亮首頁主要區塊，適合第一次使用時快速熟悉版面 |
| 使用說明 | 開啟一般說明彈窗，查詢首頁、族群頁、個股頁與顏色規則 |

```
1. 看「今日研究清單」  →  確認今天有哪些值得研究的族群
2. 點族群卡片         →  進入族群研究頁，確認強勢是否健康
3. 點個股             →  進入個股研究頁，確認在族群中的角色與風險
```

---

## 三、顏色慣例（重要）

本工具依台股慣例配色：

| 顏色 | 意義 |
|---|---|
| 紅色 | 上漲、買超、資金流入、正面訊號 |
| 綠色 | 下跌、賣超、資金流出、負面訊號 |
| 黃色 | 融資中等風險（40–70%） |
| 橙色 | 雙重籌碼警示（融資高 + 當沖高） |
| 灰色 | 中性、無明確方向 |

---

## 四、首頁

目前首頁由上到下分成四列：

| 位置 | 區塊 | 用途 |
|---|---|---|
| 第一列左側 | 盤後決策卡 | 先讀市場格局、資金主軸、風險提醒與明日觀察 |
| 第一列右側 | 多股比較圖 | 快速對照主要觀察標的的相對走勢 |
| 第二列 | 今日研究清單 | 依優先級列出值得追蹤的族群 |
| 第三列左側 | 產業地圖 / 個股研究 | 用熱力圖找族群，或切到個股篩選器 |
| 第三列中側 | 資金流向 / 法人 | 檢查流入流出、法人連買連賣與籌碼警示 |
| 第三列右側 | 今日市場新聞 | 作為背景資訊補充，桌面版會固定在右側 |

### 4-1 盤後決策卡

頁面頂部的卡片，快速告訴你今日市場的整體結構。

**市場格局**（左側）

系統依指數、廣度、資金方向自動判斷今日屬於哪種格局：

| 格局名稱 | 說明 |
|---|---|
| 擴散型上漲 | 指數漲、廣度廣、族群輪動均勻 |
| 集中型上漲 | 指數漲，但資金集中在少數族群 |
| 指數強廣度弱 | 指數漲，但上漲家數不多，少數權值撐盤 |
| 資金輪動 | 流入流出族群同時明確，資金在換位 |
| 弱勢反彈 | 指數小漲但量能弱，廣度未擴散 |
| 全面轉弱 | 指數跌、廣度低、弱勢族群增加 |
| 結構分歧 | 各指標方向不一致，需回到族群層確認 |

**資金主軸**（右側）

顯示今日資金主要流入、流出的族群，點族群名稱可直接跳轉到該族群頁。

**風險提醒**

顯示籌碼異常、廣度過度集中等需要注意的結構分歧。

**明日觀察**

今日格局下，明天最值得追蹤的方向。

---

### 4-2 多股比較圖

位於首頁第一列右側，用於快速對照多個觀察標的的相對走勢。它是盤後決策卡旁的輔助視圖，先提供價格脈絡，再往下進入研究清單與產業地圖。

---

### 4-3 今日研究清單

今日值得研究的族群，依五個類別分類呈現：

| 類別 | 意義 |
|---|---|
| 今日焦點 | 強度第一的強勢族群，優先研究 |
| 重點觀察 | 其他符合強勢條件的族群 |
| 資金轉強 | 資金剛開始流入但技術面尚未跟上，觀察是否形成方向 |
| 風險升高 | 融資或當沖偏高的族群，短線風險需留意 |
| 弱勢觀察 | 持續弱勢、法人連續賣出的族群 |

每張卡片顯示：
- **族群名稱 + 研究結論**（一句話）
- **支撐數據**（法人、廣度、強度等）
- **待確認**（接下來要看什麼）
- 「查看族群資料 →」按鈕（風險升高類型除外，因為不建議直接追進）

---

### 4-4 產業地圖（熱力圖）

視覺化顯示各族群的表現。

**格子大小**：代表成交占比（越大的格子，今日成交越活躍）  
**格子顏色**：依右上角切換的指標顯示（紅強綠弱）  
**點擊格子**：進入該族群研究頁

**四種指標模式**：

| 模式 | 說明 |
|---|---|
| 漲跌 | 各族群今日漲跌幅，觀察哪些族群領漲或領跌 |
| 廣度 | 各族群上漲家數比例，觀察漲勢是否廣泛擴散 |
| 法人 | 各族群法人今日買賣超金額，觀察資金流向 |
| 強度 | 各族群綜合強度評分，結合價格、廣度與資金 |

**右上角圓點**代表族群類別（不影響格子顏色）：
- 紅點：強勢族群
- 橙點：資金入但技術面弱
- 黃點：技術強但法人資金未跟上
- 綠點：弱勢族群

---

### 4-5 個股研究（篩選器）

點熱力圖右上角「個股研究」切換到個股篩選模式，可跨族群篩選個股，依漲跌、相對強弱、法人流向等條件排序。

---

### 4-6 資金流向 / 法人

顯示：
- **資金集中度**：資金是否過度集中在少數族群
- **流入 / 流出 Top5**：今日資金流入最多 / 流出最多的族群
- **法人連買 / 連賣**：持續連續買超或賣超的族群

---

### 4-7 今日市場新聞

顯示今日市場新聞摘要，作為研究背景補充。新聞欄不取代市場結構、資金流向或族群資料，使用時建議放在最後確認。

---

## 五、族群研究頁

從首頁點任一族群進入。

### 5-1 頂部資訊列

顯示族群名稱、今日漲跌幅、族群類別、成交占比、法人買賣超、上漲家數比例（廣度）。

麵包屑：`總覽 / 族群名稱`，點「總覽」返回首頁。

---

### 5-2 族群研究摘要

進入頁面後最先看到的結論卡，回答三個問題：

| 欄位 | 說明 |
|---|---|
| 狀態 | 這個族群今天整體狀況（資金、廣度） |
| 結構 | 上漲家數比例與法人買賣超具體數字 |
| 待確認 | 接下來最重要的驗證點 |

---

### 5-3 族群走勢圖

顯示近 20 日走勢：
- **彩色線**：本族群漲跌幅
- **灰虛線**：加權指數（基準）
- **半透明色柱**：族群成交量

顏色依族群類別（強勢=紅、資金轉強=橙、技術強=黃、弱勢=綠）。

---

### 5-4 族群分析（左欄）

| 區塊 | 說明 |
|---|---|
| 上漲家數比例 | 廣度指標，越高代表漲勢越廣 |
| 個股分類 | 族群內領漲、補漲、轉弱、弱勢各幾支 |
| 整體資金買賣超 | 近 20 日每日的資金流入流出長條圖 |
| 三大法人 | 外資、投信、自營商各自買賣超 |
| 籌碼警示 | 融資使用率或當沖比異常的個股 |

---

### 5-5 個股表格

顯示族群內所有成分股，支援三個 Tab：

| Tab | 顯示內容 |
|---|---|
| 今日概覽 | 漲跌幅、相對強弱、法人買超、大單集中度、法人連買天數 |
| 區間表現 | 近週 / 近月 / 近季累計漲幅 |
| 籌碼風險 | 融資使用率、券資比、當沖比 |

點擊列標題可排序。點個股名稱進入個股研究頁。

---

## 六、個股研究頁

從族群頁的個股表格點入。

### 6-1 頂部資訊列

顯示：個股名稱、代碼、今日漲跌幅、族群角色 badge  
麵包屑：`總覽 / 族群名稱 / 個股名稱`

---

### 6-2 個股研究摘要（右欄頂部）

最重要的結論區塊，固定回答四個問題：

| 欄位 | 說明 |
|---|---|
| 角色 | 在哪個族群裡扮演什麼角色（領漲、補漲、轉弱、弱勢） |
| 優勢 | 目前支持這檔股票的主要因素 |
| 風險 | 目前最需要注意的籌碼或結構問題 |
| 待確認 | 接下來最重要的觀察點 |

這四行不包含建議，只呈現客觀事實與待驗證項目。

---

### 6-3 四面向評估（左欄）

從四個角度快速評估個股目前狀態：

| 面向 | 評估邏輯 |
|---|---|
| 趨勢 | 相對強弱（RS）與近週 / 近月 / 近季表現 |
| 籌碼 | 法人方向、連買天數、大單集中度 |
| 評價 | P/E、P/B 位階（相較歷史） |
| 成長 | ROE、EPS 年增率、營收年增率 |

每個面向顯示評分（強 / 支持 / 偏低 / 改善…）與說明一句。  
下方自動產生總結句：例如「整體偏強，主要由籌碼支持與成長改善帶動，趨勢尚需確認。」

---

### 6-4 族群內角色（左欄）

顯示這檔股票在族群中的定位：

| 欄位 | 說明 |
|---|---|
| 角色 | 領漲股 / 補漲觀察股 / 轉弱觀察股 / 弱勢股 |
| 族群排名 | 在同族群中依相對強弱的排名 |
| 相對強弱 | 與族群平均的強弱差（正值 = 強於族群） |
| 法人買賣超 | 今日法人買賣超金額，含連買 / 連賣天數 |
| 大單集中度 | 0–100，越高代表大戶籌碼越集中 |

---

### 6-5 籌碼風險（左欄）

| 指標 | 說明 | 警戒值 |
|---|---|---|
| 融資使用率 | 融資占可融資額度比例 | ≥ 70% 偏高 |
| 券資比 | 融券 / 融資比例 | ≥ 30% 注意 |
| 當沖比 | 當沖成交量占總成交量 | ≥ 45% 偏高 |

系統自動判斷整體風險等級（低 / 中 / 偏高 / 高），並產生一句說明。

---

### 6-6 基本面（左欄）

顯示 P/E、P/B、ROE、EPS 年增率、營收年增率，每個數值標示目前位階（偏低 / 中 / 偏高）。

---

### 6-7 價格走勢與成交量（右欄）

K 線圖，可切換 20 日 / 60 日。

- **K 線**：紅漲綠跌
- **灰線**：成交量趨勢（絕對量）
- **色柱**：量差（紅色 = 量增、綠色 = 量縮）

Hover 任一 K 棒可查看當日開 / 高 / 低 / 收 / 成交量 / 量差。

---

### 6-8 法人籌碼（右欄）

頂部顯示一句客觀解讀（例如：「法人合計買超，但主力大單偏流出，籌碼結構略有分歧」）。

明細分兩組：

**法人買賣超**
- 外資 / 投信 / 自營商 / 主力（億元）

**大單與信用**
- 大單差（大單買入 - 大單賣出）
- 融資餘額（張）
- 融券餘額（張）

## 七、常見問題

**Q：資料是即時的嗎？**  
A：目前為模擬資料，實際串接後將為盤後更新（每日收盤後）。

**Q：紅色是漲還是跌？**  
A：紅色 = 上漲，綠色 = 下跌。這是台股慣例。

**Q：族群類別怎麼判斷？**  
A：由系統根據技術面與法人資金方向自動分類：

| 類別 | 判斷邏輯 |
|---|---|
| 強勢族群 | 技術面強且法人資金同步支持 |
| 資金入但弱 | 法人資金流入，但技術面尚未跟上 |
| 技術強無資金 | 技術面強，但法人資金未同步 |
| 弱勢族群 | 技術面弱且資金持續流出 |

**Q：個股角色怎麼判斷？**  
A：依相對強弱、法人支持度自動分類：

| 類別 | 說明 |
|---|---|
| 領漲股 | 漲幅與 RS 優於族群平均，法人或大單具支持 |
| 補漲觀察股 | 跟隨族群上漲，但 RS 或法人支持低於領漲股 |
| 轉弱觀察股 | 開始落後族群，需觀察是否持續 |
| 弱勢股 | 明顯落後族群，資金無支持 |
